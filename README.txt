to run these tests, please follow these steps

INSTALLATION
============
1. create a web server (e.g. http://kvwebmetests/) and set the directory ./tests as its DOCUMENT_ROOT
2. create another web server (e.g. http://kvwebmerun/) and set ./run/trunk as its DOCUMENT_ROOT
3. make sure that ./run is empty, and is writable by the webserver (chmod 777 run)
4. check the file tests/config.php to make sure the settings look correct
5. go to http://kvwebmetests/ in your browser!
